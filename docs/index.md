# Engenharia de Transportes e Mobilidade

Bem-vindo ao **Repositório Acadêmico CETMob**, um hub centralizado de conhecimento mantido pelos próprios estudantes do Curso de Engenharia de Transporte e Mobilidade (UERGS/UFPEL).

Este projeto é *Open Source* e tem como objetivo democratizar o acesso aos materiais de estudo, resumos e ferramentas para os futuros Engenheiros.

## O que você encontra aqui? (Atualmente)

* **'Wiki' das Disciplinas:** Resumos de aulas, materiais e dicas das disciplinas já cursadas pelos estudantes.

## O que vem por aí? (Futuramente) 

* **Acervo de Trabalhos Acadêmicos:** Biblioteca com os trabalhos dos veteranos, podendo ser TCCs,  projetos de extensão ou trabalhos das disciplinas.
* **Projetos e Códigos:** Repositório de softwares e scripts úteis.
* **Banco de Questões:** Listas de exercícios resolvidas pelos alunos.
* **Laboratório de Dados:** Notebooks com análises de tráfego e simulações.
